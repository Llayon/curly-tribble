# ADR 0002: Асинхронная Реактивная Навигация (Zenith Navigator)

* **Статус**: Accepted
* **Дата**: 2026-05-07
* **Автор**: Savage Fantasy Agent

## Контекст (Context)
В Savage Fantasy персонажи (колонисты) должны эффективно находить путь в динамически меняющемся мире. Традиционный синхронный поиск пути (A*) вызывает просадки FPS при массовых запросах. Кроме того, необходимо учитывать разную проходимость ландшафта (дороги, болота).

## Решение (Decision)
Внедрена система "Zenith Navigator", состоящая из:
1. **Integer-Scaled Grid**: Карта проходимости хранит веса `u8`. Базовая стоимость земли — 20, дороги — 10, блокеры — 0. Это обеспечивает высокую точность без f32.
2. **Async Pathfinding**: Расчеты выполняются в `AsyncComputeTaskPool`. Результат возвращается как `Option<Vec<Vec3>>`.
3. **Reactive Invalidation**: При добавлении препятствия (`NavObstacle`) рассылается событие `PathBlockEvent { cell: IVec2 }`. Все активные пути, пересекающие эту клетку, помечаются как недействительные.
4. **Commands Extension**: Реализована команда `commands.move_to(target)`, которая инкапсулирует весь пайплайн (поиск цели -> запуск задачи -> ожидание результата).

## Обоснование (Rationale)
Использование целых чисел повышает производительность кэша. Асинхронность гарантирует стабильные 60+ FPS. Реактивность через Observers убирает необходимость в дорогостоящем поллинге препятствий каждый кадр.

## Последствия (Consequences)
* **Положительные**: Плавный геймплей без фризов. "Умное" поведение (предпочтение дорогам). Мгновенная реакция на строительство новых стен.
* **Отрицательные**: Необходимость обработки состояния `ComputingPath` в ИИ (персонаж не идет мгновенно).
* **Нейтральные**: Все расчеты сетки должны проходить через хелпер `world_to_grid`.
* **Ограничения (Constraints)**: Если сетка 1x1, дистанция к цели (`Interaction Radius`) в запросе поиска пути не может быть меньше `1.0` метра, так как центр ближайшей соседней клетки находится ровно на дистанции 1.0м. Алгоритм не найдет путь, если задать `radius < 1.0` к центру заблокированной клетки.
